Términos de servicio

Última actualización: 2026

1. Uso de la aplicación

Universal Profile Manager es una aplicación gratuita de uso personal. Al usarla, aceptás que es tu responsabilidad mantener segura tu contraseña maestra. El desarrollador no tiene acceso a tus datos en ningún momento.

2. Sin garantías

La aplicación se distribuye tal como está, sin garantías de ningún tipo. El desarrollador no se hace responsable por pérdida de datos derivada de olvido de la contraseña maestra, desinstalación, o borrado manual de archivos.

3. Propiedad intelectual

Este software se distribuye como aplicación compilada. No está permitida la redistribución de copias modificadas ni la ingeniería inversa del código compilado.

4. Servicios de terceros

La aplicación utiliza los servicios de autenticación oficiales de GitHub y Google. Al vincular tu cuenta de alguno de estos servicios, también aceptás sus propios términos de uso:

  • Términos de servicio de GitHub
  • Términos de servicio de Google

5. Cambios

Estos términos pueden actualizarse en cualquier momento. La versión vigente siempre estará disponible en esta página.
