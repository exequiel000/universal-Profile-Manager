# Universal Profile Manager

Aplicación de escritorio para Windows que te permite guardar y organizar
perfiles de servicios de desarrollo (GitHub, GitLab, Bitbucket, Docker,
NPM, API, SSH) en un solo lugar, con vinculación opcional de tus cuentas
de GitHub y Google.

Todo se guarda **cifrado y 100% local** en tu computadora, protegido por
una contraseña maestra que solo vos conocés. Nada se envía a servidores
externos salvo las llamadas oficiales necesarias para iniciar sesión con
GitHub o Google.

## Descarga

👉 **[Descargar la última versión](../../releases/latest)**

1. Descargá `UniversalProfileManager_Setup.exe` desde Releases
2. Ejecutalo y seguí el instalador
3. Listo — se crea un acceso directo en el Escritorio automáticamente, no
   necesitás usar la consola ni instalar nada más

## Funciones

- Guardar, buscar, editar y eliminar perfiles por tipo de servicio
- Vincular tu cuenta de **GitHub** (inicio de sesión oficial, sin
  contraseñas guardadas)
- Vincular tu cuenta de **Google** (inicio de sesión oficial, sin
  contraseñas guardadas)
- Interfaz en español/inglés según el idioma de tu Windows
- Vault local cifrado (PBKDF2 + AES) protegido por contraseña maestra

## Privacidad

Este programa no recopila ni transmite tus datos a ningún servidor del
desarrollador. Política de privacidad completa acá:
[Política de privacidad](https://exequiel000.github.io/universal-Profile-Manager/privacy.html)
