Universal Profile Manager

Universal Profile Manager es una aplicación de escritorio para Windows que te permite guardar y organizar perfiles de servicios de desarrollo (GitHub, GitLab, Bitbucket, Docker, NPM, API, SSH) en un solo lugar, cifrados localmente con una contraseña maestra que solo vos conocés.

La app permite vincular tu cuenta de GitHub y Google mediante el inicio de sesión oficial de cada plataforma (OAuth), para identificar tus cuentas conectadas sin que la aplicación almacene tu contraseña real en ningún momento.

Todos los datos se guardan exclusivamente en tu computadora. Nada se envía a servidores externos salvo las llamadas oficiales necesarias para iniciar sesión con GitHub o Google.
